# Recibo Público Digital

Mini webapp estática para presentar la propuesta ciudadana "Recibo Público Digital".

## Archivos

- `index.html`: estructura de la página.
- `styles.css`: diseño visual.
- `app.js`: interacción del selector de escenarios y modal de reporte.

## Publicación gratuita

Este proyecto puede publicarse gratis en GitHub Pages porque solo usa HTML, CSS y JavaScript.
